[MediaWiki user scripts](https://en.wikipedia.org/wiki/Wikipedia:User_scripts) developed primarily for use on English Wikipedia and other Wikimedia projects.

List of scripts ([on-wiki version](https://en.wikipedia.org/wiki/User:ClaudineChionh/Scripts)):

* [copyPageContent.js](/src/copyPageContent.js) ([wiki](https://en.wikipedia.org/wiki/User:ClaudineChionh/Scripts/copyPageContent)) – Copy the content of the current page or edit form to the system clipboard
* [highlightPiped.js](/src/highlightPiped.js) ([wiki](https://en.wikipedia.org/wiki/User:ClaudineChionh/Scripts/highlightPiped)) – Fork of [User:Jeeputer/highlightPiped](https://en.wikipedia.org/wiki/User:Jeeputer/highlightPiped) that can be used in User and Drafts namespaces as well as Main
* [SkinSwitcher.js](/src/SkinSwitcher.js) ([wiki](https://en.wikipedia.org/wiki/User:ClaudineChionh/Scripts/SkinSwitcher)) – Fork of [User:Eizen/SkinSwitcher](https://en.wikipedia.org/wiki/User:Eizen/SkinSwitcher) to choose between default desktop skins as of 2024
